Create a second visual version of the neuropsychology website. Keep all current pages, content, and structure exactly the same. Only change the visual design. Do NOT modify Version 1 (current beige look).

Version 1 stays as-is at the current GitHub Pages URL. Version 2 can be on a separate branch or subfolder so both can be compared.

## Design Direction

### Colour Scheme
- Background: Pure white (#ffffff)
- Accent colour: A warm taupe or soft sage green (NOT dark/navy — everything stays light)
- Text: Dark charcoal (#1a1a1a)
- Subtle section dividers: Very light grey (#f5f5f5) to break up sections
- No dark mode, no dark backgrounds anywhere

### Hero Section
- Full-width hero with a background video loop
- Use a free stock video from Pexels — search for "tree canopy sunlight" or "sunlight through forest" (matches his brain-tree logo concept)
- The video should be: slow, calm, bright, warm sunlight filtering through nature
- Add a subtle white-to-transparent gradient overlay on top so text is readable
- Text stays dark/charcoal (not white) since the overlay keeps the background light

### Sophistication Upgrades
- Scroll-triggered fade-in animations on sections (subtle, not flashy)
- Reveal animations as user scrolls (elements slide up gently)
- Clean, generous white space
- Premium typography — keep Manrope + Newsreader fonts, maybe slightly larger headings
- Card sections (assessment areas, therapy concerns) with soft shadows and rounded corners
- Subtle hover effects on buttons and cards
