# Neuropsychological Assessments website

Custom Next.js site for [neuropsychologicalassessments.com](https://www.neuropsychologicalassessments.com).

Brand matches the domain. Public name: **Neuropsychological Assessments**.

## Local development

```powershell
npm install
npm run dev
```

Open `http://localhost:3000` (or the port shown in the terminal).

## Verification

```powershell
npm run lint
npm run build
npm run test:e2e
```

Static export lands in `out/`.
